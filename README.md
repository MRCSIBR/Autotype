## Autotype
### Simulador de tipeo

Este script simula la escritura de texto en pantalla.
Con la variable `rand` se controla la velocidad de tipeado.
  
    rand(108..242) 

Setup: 
  
  Instalar nim version 1.6.0 desde:
  https://nim-lang.org/install.html

Para compilar: 
    
    nim c -r main.nim
